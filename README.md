## Getting Started

### Installation

1. Clone the repo
```sh
git clone https://github.com/decipher07/DeepCoin.git
```
2. Install Python packages
```sh
pip install flask pycryptodome
```
3. Run using command:
```Python
python node.py
```
4. Browse
```
localhost:5000
```
